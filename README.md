# 🐴 My Farm – מערכת ניהול חכמה לחווה

**My Farm** היא מערכת ווב מתקדמת לניהול פעילות חווה, שנועדה לייעל ולרכז את כל תהליכי הניהול במקום אחד: שיעורים, מדריכים, רוכבים, סוסים ולוח זמנים דינמי.

---

## 🎯 מטרת הפרויקט
הפרויקט פותח במטרה לפתור בעיות תפעוליות בניהול חווה:
- מניעת התנגשויות בלוח זמנים
- שיבוץ יעיל של מדריכים וסוסים
- שקיפות מלאה של פעילות יומית ושבועית
- שיפור חוויית המשתמש לצוות הניהולי

---

## ⚙️ יכולות מרכזיות
- 📅 מערכת לוח זמנים אינטראקטיבית (יום / שבוע / חודש)
- 👨‍🏫 ניהול מדריכים ותלמידים
- 🐎 ניהול סוסים ומגרשים
- 🔄 שיבוץ דינמי של משאבים
- 📊 תצוגות סיכום לפי ימים ושבועות
- ⚡ התאמה חכמה של נתונים בזמן אמת
- 🧠 לוגיקה עסקית לניהול שיעורים מורכבים

---

## 🧱 ארכיטקטורה וטכנולוגיות
- Angular (Frontend SPA)
- TypeScript
- FullCalendar (ניהול לוח זמנים מתקדם)
- REST API / Backend שירותים
- PostgreSQL (בסיס נתונים רלציוני)
- Supabase / שירותי דאטה (לפי הצורך בפרויקט)

---

## 💡 דגשים טכניים
- עבודה עם רכיבים (Component-based architecture)
- ניהול State מורכב בין ישויות (מדריך/סוס/רוכב)
- סינון דינמי לפי משתמש והרשאות
- אופטימיזציה של טעינת נתונים לפי טווח זמן
- טיפול באירועים בזמן אמת בלוח השנה

---

## 👨‍💻 פיתוח
פותח כפרויקט Full-Stack במסגרת עבודה בחברת "מח אתרים", עם דגש על UX, ביצועים וארכיטקטורה נקייה.
